# Frontend Rebuild Docs

This folder specifies the frontend rebuild for the helical pile design application.

## Files

- [rebuild_plan.md](./rebuild_plan.md) - full frontend plan for TypeScript, React, shadcn/ui, Tailwind, Three.js, editor state, routes, SSE chat streaming, math rendering, API integration, testing, and rollout.

## Frontend Goal

Rebuild the current frontend prototype into a production-quality Vercel-hosted web application that can support authenticated users, saved projects, paid exports through Stripe, AI-assisted design generation, and a robust 3D design editor backed by the AWS-hosted FastAPI API.
